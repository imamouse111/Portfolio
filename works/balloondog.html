Balloon Dog

2020

Here are some of my first renders in Blender, a massively powerful Open Source 3D modeling and animation program that I'm learning through Lynda.


First up is a more photo-realistic version of a Jeff Koons sculpture. I was messing around with a few different lighting setups for this and I thought this one, which includes a purple fill light, had a bit more visual interest. I really want to circle back to it and try using HDRs or some different world lighting. This version just has a super simple studio-style backdrop.
